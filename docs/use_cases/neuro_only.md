# Neuro and behavioral data only

[![English](https://img.shields.io/badge/Language-English-blue.svg)](neuro_only.md) [![中文](https://img.shields.io/badge/语言-中文-red.svg)](neuro_only_zh.md)

Use this path if you need **open fNIRS and behavioral CSVs** without GitHub, MP4s, or enactment scripts.

## Steps

1. Download and unzip **[Zenodo R0](https://doi.org/10.5281/zenodo.20707429)**.
2. Read **`QUICKSTART_neuro.md`** in the zip root.
3. For file formats and trial columns, see **`data/README.md`** in the package (or [GitHub](https://github.com/BAK262/mENACT/blob/main/data/README.md) · [中文](https://github.com/BAK262/mENACT/blob/main/data/README_zh.md)).
